# 사내 TODO 메모장

네트워크 없이 로컬에서 쓰는 순수 HTML/CSS/JS 할일+메모장입니다.
데이터는 지정한 **폴더 안에 여러 파일로 분산 저장**됩니다 (DRM 안 걸리는 로컬 파일).

## 접속

크롬/엣지에서 아래 주소로 접속합니다 (사내망에서 접속됩니다):

**https://jeensh.github.io/pdx-todo/**

- 첫 접속 후엔 화면 코드가 브라우저에 캐시되어 **네트워크 없이도 열립니다**(서비스워커).
  이후에는 사내망/인터넷이 끊겨도 그대로 사용 가능합니다.
- 주소창 오른쪽 설치 아이콘으로 **앱처럼 설치**(바탕화면/시작메뉴 아이콘)할 수도 있습니다.
- **데이터는 GitHub에 올라가지 않습니다.** 각자 고른 로컬 폴더에만 저장됩니다.
- (대안) 인터넷 없이 완전 로컬로 쓰려면 폴더째 받아 `index.html`을 더블클릭해도 됩니다.

## 사용법

1. 위 주소로 열거나 `index.html`을 크롬/엣지로 엽니다.
2. 처음 실행하면 **메모 폴더를 선택**하라는 화면이 뜹니다 → 로컬 디스크에 폴더를 하나 지정.
3. 이후 모든 메모가 그 폴더 안에 저장됩니다. 다시 열 때 권한 팝업에서
   **"매번 방문 시 허용"**을 고르면 다음부터 자동 연결(폴더당 한 번, 파일별 아님)입니다.

- **할일**: 상단 입력창에 쓰고 Enter. 체크로 완료, ✕로 삭제
- **보기**: 오늘 / 이번 주(월~일) / **달력** / 전체 / 완료됨 / 그룹별 — 할일마다 그룹과 날짜를 붙일 수 있어
  날짜별·그룹별로 동시에 관리됩니다. 기한 지난 미완료는 "밀린 할 일"로 표시
- **달력**: 월 달력에서 날짜별 할일을 한눈에 보고, 칩을 누르면 그 메모로 이동.
  빈 날짜를 누르면 그 날짜로 새 할일이 추가됩니다. ◀ ▶로 달 이동, 오늘 버튼으로 이번 달 복귀
- **그룹**: 사이드바 "＋ 새 그룹". 그룹 삭제 시 소속 할일은 미분류로 이동
- **검색**: 상단 검색바(Ctrl+F). 제목·본문에서 부분일치(대소문자 무시)로 찾고 일치 부분을 강조.
  · 범위 토글(전체/제목만/본문만) · 여러 조각은 `and`로 연결하면 모두 포함(AND) 검색
  · 본문 매칭은 주변 스니펫 미리보기, 클릭하면 메모를 열고 그 위치를 강조
  · 결과에 그룹·완료 포함 필터, "더보기"로 이어보기. ESC로 종료.
  · 성능: 본문 텍스트를 캐시하고 수정된 항목만 다시 계산 — 문서가 많아도 빠름
- **메모**: 할일을 클릭하면 우측에 에디터
  - 서식: 굵게/기울임/밑줄/취소선/소제목/목록/구분선
  - **이미지**: 붙여넣기(Ctrl+V), 드래그, 툴바 버튼 — JSON 안에 내장(큰 이미지는 자동 축소).
    이미지 클릭 후 Delete로 삭제
  - **표**: 툴바 ⊞ 표 → 표 안에 커서를 두면 편집 바가 뜸.
    ＋행 ＋열 −행 −열 · **병합→**(오른쪽 셀과 병합) **병합↓**(아래 셀과 병합) **해제**(병합 풀기) · Tab으로 셀 이동.
    엑셀에서 표를 복사해 붙여넣으면 편집 가능한 표로 들어옵니다(병합 셀·서식 포함).
    · **셀 배경색**: 셀을 드래그로 선택하거나 커서를 둔 뒤 색 팔레트 클릭
    · **드래그 다중 선택 → 병합**: 여러 셀을 드래그해 선택하고 "병합"으로 한 칸으로 합침, "해제"로 되돌림
  - **영역 크기 조절**: 가운데 목록과 오른쪽 메모 사이 경계를 드래그하면 폭이 조절됩니다(이 기기에 저장).
  - **링크**: 툴바 🔗 링크 →
    · **할일 연결** — 다른 할 일을 골라 링크. 클릭하면 그 할 일로 이동(제목이 바뀌면 링크도 자동 갱신)
    · **문서·웹 주소** — URL이나 파일 경로(`\\서버\공유\문서.docx`, `C:\...`) 입력. 클릭하면 열립니다.
    링크를 편집하려면 Alt+클릭.
  - **링크 복사**: 메모 상단 "🔗 링크 복사" → 현재 할 일의 링크가 복사됩니다.
    다른 메모에 붙여넣으면 이 할 일로 연결되는 링크가 생깁니다.

## 이름 설정

좌측 상단 제목을 클릭하면 이름을 입력할 수 있고, **"○○○의 메모장"**으로 바뀝니다
(이름도 index.json에 함께 저장됩니다).

## 저장 (폴더 분할 저장)

자동 저장이지만, 상단 우측 **"저장"** 버튼(또는 Ctrl+S)으로 즉시 저장할 수도 있습니다.
저장 상태는 그 옆에 "저장됨 HH:MM"(초록)으로 표시됩니다.


선택한 폴더 안에 이렇게 나뉘어 저장됩니다:
```
내 메모/
├─ index.json        ← 목록(제목·그룹·날짜·완료 등 메타데이터)
├─ notes/<id>.html   ← 각 메모 본문
└─ images/<해시>.png  ← 이미지 파일 (내용 해시로 이름, 중복 자동 제거)
```

- **편집한 메모 파일 하나만** 저장 → 문서가 수천 개여도 저장이 빠릅니다.
- 시작할 때 `index.json`만 읽어 목록·달력이 즉시 뜨고, 본문·이미지는 열 때 로딩합니다.
- 폴더를 통째로 복사해 다른 PC로 옮길 수 있습니다. 파일들이라 백업·확인이 쉽습니다.
- **로컬 디스크 권장** — 네트워크 공유폴더(`\\서버\...`)는 파일이 많으면 느릴 수 있습니다.
- 상단 우측 폴더 이름을 누르면 다른 폴더로 바꿀 수 있습니다.
- `index.json`이 손상된 경우 앱은 **덮어쓰지 않고** 알림만 표시합니다(백업으로 복구 가능).

## 파일 구성

| 파일 | 역할 |
|---|---|
| `index.html` | 화면 구조 |
| `style.css` | 디자인 (외부 폰트/CDN 없음) |
| `app.js` | 할일·에디터·저장 로직 |
